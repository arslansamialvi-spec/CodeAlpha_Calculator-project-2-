# Basic Calculator

A simple and responsive web-based calculator that performs basic arithmetic operations. This project is built using HTML, CSS, and JavaScript and provides an easy-to-use interface for performing everyday calculations.

## Features

- Addition (+)
- Subtraction (-)
- Multiplication (×)
- Division (÷)
- Decimal number support
- Clear (C) button to reset calculations
- Responsive and user-friendly interface
- Real-time display of input and results

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

## How to Use

1. Open the calculator in your web browser.
2. Click the number buttons to enter values.
3. Select the desired arithmetic operator.
4. Press the **=** button to display the result.
5. Use the **C** button to clear the screen and start a new calculation.
